### logs表

| 字段 |     _id      | _openid          | content  | fileIDs              | time          |
| ---- | :----------: | ---------------- | -------- | -------------------- | ------------- |
| 说明 | 区别一条记录 | 用户的唯一身份码 | 日志内容 | 附件在云存储中的链接 | 日志时间      |
| 类型 |     str      | str              | str      | array，元素为str     | str           |
| 备注 |   自动生成   |                  |          |                      | 例如“8月23日” |

| is_delete        |ifread      |ifstar      |      |      |      |
| ---------------- | ---- | ---- | ---- | ---- | ---- |
| 是否删除         |是否已读      |是否有星      |      |      |      |
| boolean          |boolean      |boolean      |      |      |      |
| 1表示删除，0反之 | 1表示删除，0反之| 1表示删除，0反之|      |      |      |
